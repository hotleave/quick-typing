# 版本历史

本程序所有变更历史都将记录于本文档，方便大家了解更新内容及规划。

该文档基于[更新日志](https://keepachangelog.com/zh-CN/1.0.0/)编写，版本号遵循[语义化版本](https://semver.org/)

## 待发布

### 新特性

- 内置常用字及文章
- 文件载文
- 速度练习
- 词库练习
- 看打模式
- 对接小折接口
- 成绩统计

## [1.0.1] - 2021-03-12

### 新特性

- 成绩中增加小拆哈希码，默认不选择
- 成绩中增加暂停时长及次数，默认不选择

## [1.0.0] - 2021-03-10

### 新特性

- 手动剪切板载文
- 赛文跟打
- 用是否加粗方式进行词语提示
- 码长提示，用不同颜色代表相应码长的词语
- 编码提示，包括单字及词语
- 选重提示及标点顶屏提示
- 跟打成绩自动复制到剪切板
- 按键统计图
